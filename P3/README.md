## ⚙️ Setup

### 4.1 Instalación

Sigue estos comandos en tu terminal (Windows PowerShell preferido):

```bash
# Crear entorno virtual
python -m venv venv

# Activar entorno virtual
.\venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt
```
